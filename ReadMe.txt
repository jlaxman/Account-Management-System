#README: How to Run the Account Management System
Running the Program
To run the Account Management System, follow these simple steps:

Compile the Source Code:

Open your terminal or command prompt.
Navigate to the directory containing the project files.
Use the g++ compiler to compile the program with the following command:

g++ -std=c++17 main_sailaxman.cpp BankAccount_sailaxman.cpp StockAccount_sailaxman.cpp Account_sailaxman.cpp Node_sailaxman.cpp Sort_sailaxman.cpp -o main

Execute the Compiled Program:

Run the compiled program using the command:

./main

Interact with the Program:

Once the program starts, you will be greeted with a welcome message and options to access different accounts (Stock Portfolio Account, Bank Account, or Exit).
Follow the on-screen instructions to navigate through the program.
You can view account balances, buy or sell shares, view transaction history, and much more.


For more details on the project, please refer to the provided project report.
